MyDiary

This is a "basic" diary that allows you to digitally write entries and view them.From a normal user perspective the GUI might make the user doubt its capabilities, but don't let the looks deceive you, because this is my best work thus far.What makes this a masterpiece from its counterparts is how user data is created and abstracted. User data is protected and stored in PNG images, the image it self is not encrypted but the data inside the image is encrypted.If you open any one of the created PNG images, they look like blank images!.These "normal" looking images protect your information from prying eyes.The simplicity and elegance of my program stores your data(password and entry) locally and hidden from everyone,including you.The only downside to my program is that it uses a 3-Number PIN for encryption,this downside will be fixed in the next release.
